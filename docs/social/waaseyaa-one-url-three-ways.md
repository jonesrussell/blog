## LinkedIn

The payoff of my last few build videos: one URL on Waaseyaa serving the same content three ways.

A normal web page for people. Clean Markdown when an AI agent asks for it, via HTTP content negotiation. And a public read-only MCP server an agent can query directly, anonymous and enforced at three layers so nothing writable ever leaks.

No API written for any of it. It's in the framework.

The content is a public-domain Anishinaabe legend, the origin of the five totems from William W. Warren's 1885 History of the Ojibway Nation, captured live from Wikisource into the app. Waaseyaa is my sovereignty-first, agent-readable open-source PHP framework: one process, SQLite, small enough to run on a Raspberry Pi, built so a community can own and self-host its own content.

Still alpha, still built in the open. The read side works cleanly; getting content in is still rough and I left that in the take.
