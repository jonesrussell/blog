## LinkedIn

Waaseyaa is a PHP CMS framework forked from Drupal 11's core and shaped by Laravel 12's developer experience. It preserves Drupal's entity/field model and access control while adopting clean service providers, expressive routing, and modern PHP 8.4+ practices on Symfony 7 components. The result is 43 packages across seven architectural layers with a Nuxt 3 admin SPA. The first application built on it is Minoo, a platform for managing and sharing Indigenous knowledge, live at minoo.live. This is the first post in a six-part series covering the framework's architecture and how structured AI collaboration with GitHub issues and codified context made it possible to build solo. https://jonesrussell.github.io/blog/waaseyaa-intro/
