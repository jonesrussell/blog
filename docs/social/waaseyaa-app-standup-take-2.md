## LinkedIn

Take two of standing up a brand-new app on Waaseyaa, and this time it finished clean.

Waaseyaa is the sovereignty-first PHP framework I'm building. This run was fully agent-driven, no manual terminal work: the agent ran create-project, found the dev server command, started it in the background, opened the browser, and confirmed a stock app was live at localhost:8080, greeting you with Ahnii.

A few things I left honest in the video:

I'm running it in Codex, not my usual Claude Code setup, so the agent goes in cold with no Waaseyaa knowledge. It tripped once on a leftover directory from the last take, deleted it, and reran. Most of the remaining time was just Composer downloading packages. It came in around half the time of take one.

One PHP process, SQLite, no Docker. Small enough to run on a Raspberry Pi a community can physically own.

Next run: warm the package cache and hand the agent the framework's own skills so it already knows the commands. That's the version where this gets fast and boring, which is the goal.

Still alpha, built in the open.
