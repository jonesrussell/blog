# Hugo blog shortcodes: adding a visual component system to PaperMod

Blog URL: https://jonesrussell.github.io/blog/hugo-blog-visual-shortcodes/

## Facebook

PaperMod is a great Hugo theme, but it ships with no component library. No callouts, no numbered steps, no before/after panels. I got tired of hacking blockquotes and wrote six shortcodes to fix that.

One Claude Code session later: callout, steps, stats, compare, cta, and pullquote — all hooked into PaperMod's CSS variables so dark mode just works. Nine files total, no JavaScript.

The new post walks through each shortcode with the actual template code and usage examples.
