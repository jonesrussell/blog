## LinkedIn

The agent-readable web is usually a theory. Here it is on a site that matters.

rhtcircle.ca is the Robinson Huron Treaty resource hub, built on the Waaseyaa framework. It now serves every page two ways: the normal human view, and a machine-readable Markdown layer reachable with a format switch, plus a /llms.txt index so an AI agent can discover the whole site.

Why this matters for a treaty site specifically. The people who need this information often arrive through a chatbot or a search assistant, not a polished web page. If the assistant gets a clean Markdown version and a map of what exists, it can answer a real question, point to housing help, education funding, or a records request, and cite the actual page instead of guessing.

One URL, two readers. Same content, no separate API to maintain, no duplicated copy to drift. The human page and the agent feed come from the same source.

This is the part of Waaseyaa I am most glad to see in production: making a community's information legible to the tools people actually use to reach it.

See the index: https://rhtcircle.ca/llms.txt
